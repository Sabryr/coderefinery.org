A Nordic e-Infrastructure Collaboration Project

About the project

The goal of this project is to establish a software development e-infrastructure coupled with necessary technical expertise and extensive training activities in order to address the growing needs of computational communities. This is a project within the Nordic e-Infrastructure Collaboration (NeIC). NeIC is an organisational unit under NordForsk.

Training

Train Nordic research groups and projects in state-of-the-art tools and practices in modern collaborative software engineering to build modular, reusable, maintainable, sustainable, reproducible, and robust software.

Consulting and infrastructure

Assist groups in migrating from ad-hoc in-house infrastructure to collaborative industry-strength community solutions encouraging code review and reproducible code.

Visibility and impact

Increase visibility of Nordic software projects by creating a platform where software can be shared.

Community

Building a community by aggregating and providing courses and organizing training, summer schools, meet-ups, tutorials and forums.
