About the project

The goal of this project is the establishment of a software development e-infrastructure coupled with necessary technical expertise and extensive training activities in order to address the growing needs of computational communities. This is a project within the Nordic e-Infrastructure Collaboration (NeIC). NeIC is an organisational unit under NordForsk.

Training

Train Nordic research groups and projects in state-of-the-art tools and practices in modern software development.

Consulting and infrastructure

Assist groups in migrating from ad-hoc in-house infrastructure and solution to collaborative industry-strength community solutions encouraging code review and test-driven development.

Visibility and impact

Increase visibility of Nordic software projects by creating a platform where software can be shared.

Maintainable software

Train groups in building modular and maintainable code.

Reproducible science

Encourage open science and enable reproducible science.

Community

Building a community by aggregating and providing courses and organizing training, summer schools, meet-ups, tutorials and forums.
